## Deskripsi Singkat Aplikasi

Aplikasi ini adalah sistem CRUD (Create, Read, Update, Delete) sederhana untuk mengelola data kategori menggunakan PHP native dan MySQL. Fitur utama meliputi menampilkan daftar kategori, menambah data baru, mengubah data, dan menghapus data.

## Cara Instalasi dan Menjalankan Aplikasi

1. Pastikan XAMPP sudah terpasang (Apache dan MySQL).
2. Salin project ke folder htdocs:
	- `C:/xampp/htdocs/uts_smt4`
3. Jalankan Apache dan MySQL dari XAMPP Control Panel.
4. Buat database MySQL sesuai konfigurasi di file `config/database.php`.
5. Sesuaikan nilai koneksi database jika diperlukan (host, user, password, dan nama database).
6. Buka aplikasi melalui browser:
	- `http://localhost/uts_smt4`
